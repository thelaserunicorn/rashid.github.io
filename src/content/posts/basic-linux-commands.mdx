The terminal can feel intimidating at first, but these commands will help you navigate and manage your system with confidence.

## Navigating the File System

### `pwd` - Print Working Directory

Shows your current location in the filesystem.

```bash
pwd
# Output: /home/username/Documents
```

### `ls` - List Files

Display files and folders in your current directory.

```bash
ls
# Basic listing

ls -la
# Detailed listing with hidden files
```

### `cd` - Change Directory

Move between folders.

```bash
cd Documents
# Go into Documents folder

cd ..
# Go back one level

cd ~
# Go to home directory

cd /
# Go to root directory
```

## File Operations

### `cp` - Copy Files

```bash
cp source.txt destination.txt
# Copy a single file

cp -r folder/ newfolder/
# Copy a directory recursively
```

### `mv` - Move or Rename

```bash
mv oldname.txt newname.txt
# Rename a file

mv file.txt Documents/
# Move file to Documents folder
```

### `rm` - Remove Files

```bash
rm unwanted.txt
# Delete a file

rm -r folder/
# Delete a directory and its contents

rm -rf folder/
# Force delete (no confirmation)
```

## Reading Files

### `cat` - Display File Contents

```bash
cat filename.txt
# Display entire file

cat file1.txt file2.txt
# Display multiple files
```

### `less` - View File Page by Page

```bash
less largefile.txt
# Scroll through file with arrow keys
# Press q to quit
```

### `head` and `tail` - View File Start or End

```bash
head filename.txt
# First 10 lines

head -n 20 filename.txt
# First 20 lines

tail filename.txt
# Last 10 lines

tail -f logfile.txt
# Watch a log file in real-time
```

## Searching

### `find` - Find Files

```bash
find . -name "*.txt"
# Find all .txt files in current directory

find /home -name "document"
# Search from a specific location
```

### `grep` - Search Within Files

```bash
grep "error" logfile.txt
# Find lines containing "error"

grep -r "text" folder/
# Search recursively in a folder

grep -i "hello" file.txt
# Case-insensitive search
```

## Permissions

### `chmod` - Change Permissions

```bash
chmod 755 script.sh
# Make a script executable

chmod +x script.sh
# Add execute permission

chmod 600 private.txt
# Owner read/write only
```

### `chown` - Change Ownership

```bash
sudo chown user:group file.txt
# Change file ownership
```

## System Information

### `ps` - View Processes

```bash
ps aux
# Show all running processes

ps aux | grep firefox
# Find specific process
```

### `top` - Task Manager

```bash
top
# Interactive process viewer
# Press q to quit
```

### `df` - Disk Space

```bash
df -h
# Show disk usage in human-readable format
```

## Network

### `ping` - Test Connection

```bash
ping google.com
# Test internet connection
```

### `curl` - Fetch URLs

```bash
curl https://example.com
# Download webpage content
```

## Tips

- Use `Tab` to auto-complete commands and file names
- Use `↑` and `↓` arrows to navigate command history
- Press `Ctrl + C` to cancel a running command
- Press `Ctrl + L` to clear the terminal screen

Start with these commands and gradually explore more. The terminal becomes more powerful the more you use it.
